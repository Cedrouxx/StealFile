1. インストール方法

================================================================================

1.0. 必要PC動作環境

重要！ 本製品は64ビットOSのみサポートしています！
       Windows® 7の場合は Service Pack 1 以上が必要です。

1.0.1 必要最低動作環境:

  OS:			Windows 7(64ビット)/Windows 8(64ビット)/Windows 10(64ビット)
 
  CPU:			Intel Core i3 560 @ 3.3GHz以上
                    	AMD Phenom II X4 945 @ 3.0Ghz以上
 
  メモリ:           	6GB
  
  グラフィックカード:	NVIDIA GeForce GTX 460、AMD Radeon HD 5870
                    	およびDX11対応のグラフィックカード(1GB VRAM)
		    	詳細は1.0.3の動作確認リストをご覧ください
                    
  DVD-ROM:          	二層式DVD-ROMVD-ROM
  
  HDD 空き容量: 	30 GB
  
  サウンドカード:       DirectX 9.0c対応サウンドカードと最新のドライバー
  
  インターネット:	マルチプレイヤーモードにはブロードバンド接続のインターネットサービスが必要です

  Windows対応のマウスが必要です

1.0.2 推奨動作環境:

  OS:			Windows 7 SP1(64ビット)/Windows 8(64ビット)/Windows 10(64ビット) 
                    
  CPU:        		Intel Core i5-2500K @ 3.3GHz以上
                      	AMD FX-8120 Eight-Core @ 2.6Ghz以上
                    
  メモリ:              	8 GB
  
  グラフィックカード:   NVIDIA GeForce GTX 670、またはAMD Radeon HD 7970 / R9 280x(2GB VRAM)
                    	詳細は1.0.3の動作確認リストをご覧ください
                    
  DVD-ROM:          	二層式DVD-ROM
  
  HDD 空き容量: 	47 GB
  
  サウンド:            	DirectX® 9.0c互換の5.1チャンネル対応サウンドカードと最新のドライバー
  
  インターネット:       マルチプレイヤーモードにはブロードバンド接続のインターネットサービスが必要です

  Windows対応のマウスが必要です
  
1.0.3 リリース時の対応グラフィックカード:

NVIDIA GeForce GTX460以上、またはGT500、GT600、GT700、GT900シリーズ、Titanシリーズ 
AMD Radeon HD5870以上、またはHD6000、HD7000、R7、R9シリーズ



最新の必要最低動作環境については、弊社サポートWebサイトの本製品に関するFAQを
参照してください。http://support.ubi.com

--------------------------------------------------------------------------------
1.1. 言語設定

インストール時に選択した言語がゲーム内で使用されます。言語設定を変更するには、
ゲームを一度アンインストールしてから再インストールしてください。

--------------------------------------------------------------------------------
1.2. アンインストール 

レインボーシックス　シージはUplay PCアプリケーション内からアンインストールできます。 
Uplay PCアプリケーション内のレインボーシックス　シージのページを開き、「プレイ」
ボタンの隣にある矢印をクリックしてください。次に「アンインストール」の項目を選択し、
画面上のガイドに従ってゲームをアンインストールしてください。

2. 一般事項

================================================================================

2.1. ディスプレイ/パフォーマンス設定

レインボーシックス　シージは高解像度のテクスチャーに対応しています。テクスチャーの
質を「超高」に設定するには4GBのVRAM、「最高」にするには6GBのVRAMが必要です。

グラフィック設定はゲームのメモリ使用量に影響します。
PCのメモリが不足している場合は、それに応じてグラフィック設定を下げてください。

グラフィックカードのドライバーによっては、画面のちらつきやオブジェクトの消失が
起こる場合があります。
必ず最新のドライバーがインストールされていることを確認してください。

--------------------------------------------------------------------------------
2.2. 最適設定

初回起動時、本製品はお使いのシステムに最適な設定を検出しようと試みます。
これには数秒を要する場合があります。
本製品がお使いのコンピューターを完全に認識できなかった場合、最適設定は
「中」または「高」になります。

--------------------------------------------------------------------------------
2.3. Uplay PC

本製品はUplay PCを通してオンラインで一度限りのライセンス認証を行う必要があります。

UplayアプリケーションまたはUplay.comから新しいUplayアカウントを作成するか、
既存のUplayアカウントと本製品に付属の固有キーを使用して、ゲームとアカウントを
リンクさせてください。

一度限りのライセンス認証を行った後、オンラインおよびオフラインモードでゲームを
起動できます。オフラインモードではすべてのゲーム機能をご利用いただけません。

Uplayライセンスについては、Uplayアプリケーションのインストール ディレクトリを
参照し、data\resources\LICENSES.TXTファイルをご確認ください。

Uplay PCアプリケーションのサポートについては下記をご覧ください:
http://support.ubi.com

3. 既知の問題

================================================================================

3.1. グラフィックドライバーのインストール

グラフィックの問題が発生した場合は、ご利用中のグラフィックカード、
およびマザーボードで使用可能な最新ドライバーをインストールしてください。
主なグラフィックカードとマザーボードメーカーへのリンクは、
この文書の下部に用意されています。

--------------------------------------------------------------------------------
3.2. サウンドドライバーのインストール

サウンドに問題が発生した場合は、ご利用中のサウンドカードで使用可能な
最新のドライバーをインストールしてください。
主なサウンドカードのメーカーへのリンクは、この文書の下部に用意されています。

--------------------------------------------------------------------------------
3.3. DirectXのインストール

レインボーシックス　シージを適切に実行するには、2010年6月以降の
DirectX ランタイム ライブラリが必要です。DirectXの最新版は、下記から入手できます:
 
http://www.microsoft.com/ja-jp/download/default.aspx
 
またDirectXはインストール ディスクの「Support\DirectX」フォルダー内に含まれており、
レインボーシックス　シージのインストール時に自動でインストールされます。
 
もし「DLLファイルがない」というエラーが発生した場合、
DirectXをインストール ディスクから再インストールしてみてください。

--------------------------------------------------------------------------------
3.4. ヒント

* ゲームを、Windowsフォルダ(通常はC:\Windows内)にインストールすること、または
  Windowsシステムドライブのルート(通常はC:\)にインストールすることは
  推奨しません。

* ゲームをインストールする時は、他の実行中のプログラムを終了することを推奨します。

* グラフィックの破損が発生した場合、コントロールパネルで
  グラフィックドライバーがアプリケーション設定を上書きしない設定にしてください。
  
* 振動機能には、Xbox 360 または PlayStation®4のコントローラーを使用時にのみ
  対応します。

4.1. テクニカルサポート

日本語でのお問い合わせは、下記ウェブサイトにて受け付けております：
http://ubisoft.co.jp/support/

よくあるご質問、サーバーメンテナンスやパッチ配信など、
本製品のリリース以降の最新情報もご確認いただけます。

お使いのPCが最低動作環境を満たしているかご確認ください。PCがこの基準を
満たしていない場合、サポート担当者がお客様のサポートを行えません。

テクニカルサポートにご連絡いただく際は、次の情報をメッセージに含めるか
お手元にご用意ください:

-正式な製品名(バージョン番号を含む)
-エラーメッセージが表示された場合はその正確な内容と、発生した問題の簡単な説明
-CPU速度と製造元
-メモリ容量
-OS(オペレーティングシステム)
-お使いのグラフィックカード名とそのメモリ容量
-CD-ROMまたはDVD-ROMドライブの形式と速度
-お使いのサウンドカードの種類

返品ポリシー

テクニカルサポートへご連絡いただくまでは、弊社まで直接本製品を送らないようお願い
いたします。返品については、製品をご購入いただいた小売店やオンラインストアへ
ご相談ください。ゲームディスクが破損、または傷ついている場合は、弊社サイトのFAQで
該当するゲームに関する最新の交換手順や値段をご確認ください。
事前にご相談いただき、サポート担当者から商品返品確認番号を得ている場合を除いて、
一方的に返品されたゲームや交換の要望は受け付けておりません。
