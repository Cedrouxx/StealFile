1. Instructions d'installation

================================================================================

1.0. Configuration

TRÈS IMPORTANT ! Ce produit ne fonctionne que sur les systèmes d'exploitation 64 bits !
                Assurez-vous que Windows® 7 possède le Service Pack 1 (ou supérieur).

1.0.1 Configuration minimale :

  Système d'exploitation : Windows 7 64 bits, Windows 8 64 bits, Windows 10 64 bits.
 
  Processeur :      Intel Core i3 560 @ 3,3 GHz ou supérieur,
                    AMD Phenom II X4 945 @ 3,0 GHz ou supérieur
 
  RAM :             6 Go
  
  Carte graphique : NVIDIA GeForce GTX 460, AMD Radeon HD 5870 et cartes DX11 avec 1 Go de mémoire vidéo
                    Voir la liste des cartes compatibles 1.0.3 pour en savoir plus.
                    
  DVD-ROM :         DVD-ROM double couche (DL)
  
  Disque dur :      30 Go
  
  Son :             Carte son compatible DirectX 9.0c avec les derniers pilotes
  
  Internet :        Connexion Internet à haut débit nécessaire pour le mode multijoueur

  Souris compatible avec Windows nécessaire

1.0.2 Configuration recommandée :

  Système d'exploitation : Windows 7 SP1 64 bits, Windows 8 64 bits, Windows 10 64 bits. 
                    
  Processeur :      Intel Core i5-2500K @ 3,3 GHz ou supérieur,
                    AMD FX-8120 huit cœurs @ 2,6 GHz ou supérieur
                    
  RAM :             8 Go
  
  Carte graphique : NVIDIA GeForce GTX 670 ou AMD Radeon HD 7970 / R9 280x (2 Go de mémoire vidéo)
                    Voir la liste des cartes compatibles 1.0.3 pour en savoir plus.
                    
  DVD-ROM :         DVD-ROM double couche (DL)
  
  Disque dur :      47 Go
  
  Son :             Carte son 5.1 compatible DirectX 9.0c avec les derniers pilotes
  
  Internet :        Connexion Internet à haut débit nécessaire pour le mode multijoueur

  Souris compatible avec Windows nécessaire
  
1.0.3 Cartes graphiques compatibles au moment de la sortie :

NVIDIA GeForce GTX460 ou supérieure, gammes GT500, GT600, GT700, GT900, gamme Titan ; 
AMD Radeon HD5870 ou supérieure, gammes HD6000, HD7000, R7 et R9 ;



Pour la configuration minimale la plus à jour, consultez la FAQ du jeu sur notre site d'assistance : http://support.ubi.com.

--------------------------------------------------------------------------------
1.1. Sélection de la langue

La langue sélectionnée au début de l'installation sera celle utilisée en jeu. Pour changer la langue, veuillez désinstaller, puis réinstaller le jeu.

--------------------------------------------------------------------------------
1.2. Désinstallation 

Tom Clancy's Rainbow Six® Siege peut être désinstallé depuis l'application Uplay PC.
Rendez-vous sur la page de Tom Clancy's Rainbow Six® Siege dans l'application Uplay PC, puis cliquez sur la flèche à côté du bouton Jouer. Sélectionnez l'option Désinstaller et suivez les instructions à l'écran pour désinstaller le jeu.

2. Remarques générales

================================================================================

2.1. Paramètres d'affichage et de performances

Tom Clancy's Rainbow Six® Siege peut afficher des textures en haute résolution. Régler la qualité des textures sur Très élevée nécessite 4 Go de mémoire vidéo et la régler sur Ultra nécessite 6 Go de mémoire vidéo.

Les réglages graphiques peuvent avoir un effet sur l'utilisation de la mémoire dans le jeu. Si votre ordinateur ne possède que peu de mémoire vive, ces réglages seront revus à la baisse.

Certains pilotes de carte graphique peuvent faire trembler ou disparaître certains éléments en jeu. Il est important de s'assurer que les derniers pilotes graphiques disponibles sont bien installés. 

--------------------------------------------------------------------------------
2.2. Réglages optimaux

Lorsque vous lancez le jeu pour la première fois, celui-ci détectera les paramètres optimaux pour votre système. Cela peut prendre quelques secondes. Si le jeu ne parvient pas à analyser la configuration de votre ordinateur, le paramètre retenu sera Moyen ou Élevé.

--------------------------------------------------------------------------------
2.3. Uplay PC

Ce jeu nécessite d'être activé en ligne via Uplay PC.

Créez un nouveau compte Uplay dans l'application Uplay ou sur Uplay.com ou utilisez votre compte Uplay et la clé unique fournie avec le jeu pour lier le jeu à votre compte.

Après l'activation, vous pouvez lancer le jeu en mode en ligne ou hors ligne. Sachez que les fonctionnalités de jeu ne sont pas toutes disponibles en mode hors ligne.

Pour les licences Uplay, rendez-vous dans le dossier d'installation de l'application Uplay et cherchez le fichier data\resources\LICENSES.TXT

Si vous avez besoin d'aide pour l'application Uplay PC, rendez-vous sur :
http://support.ubi.com

3. Problèmes connus

================================================================================

3.1. Installer les pilotes pour carte graphique

Si vous rencontrez des problèmes graphiques, veuillez installer les derniers pilotes disponibles pour votre carte mère et votre carte graphique. Des liens vers les sites des principaux fabricants de cartes graphiques et de cartes mère sont disponibles à la fin de ce document.

--------------------------------------------------------------------------------
3.2. Installer les pilotes pour carte son

Si vous rencontrez des problèmes sonores, veuillez installer les derniers pilotes disponibles pour votre carte son. Des liens vers les sites des principaux fabricants de cartes son sont disponibles à la fin de ce document.

--------------------------------------------------------------------------------
3.3. Installer DirectX

Tom Clancy's Rainbow Six® Siege nécessite au moins la version de juin 2010 de DirectX pour fonctionner correctement. Pour obtenir la dernière version de DirectX, rendez-vous sur :
 
http://www.microsoft.com/download/
 
DirectX est également fourni sur le disque d'installation, dans le dossier Support\DirectX, et est automatiquement installé lorsque vous installez Tom Clancy's Rainbow Six® Siege.
 
Si un message d'erreur s'affiche concernant d'éventuels fichiers DLL manquants, réinstallez DirectX depuis le disque d'installation afin de résoudre le problème.

--------------------------------------------------------------------------------
3.4. Conseils

* Il n'est pas recommandé d'installer le jeu dans le dossier Windows (généralement C:\Windows), dans un sous-dossier du dossier Windows, ni à la racine de votre système (généralement C:\).

* Il est recommandé de fermer tous les programmes en cours avant de lancer l'installation du jeu, ceci afin de limiter le risque de conflits entre logiciels.

* Si vous rencontrez des problèmes graphiques, assurez-vous que le panneau de contrôle des pilotes graphiques n'est pas configuré pour écraser les paramètres de l'application.
  
* La fonction de vibration n'est compatible que lorsque vous utilisez une manette Xbox 360 ou PlayStation 4.

4. Sites Internet importants, informations de contact et service client

================================================================================

4.1. Service client

Avant de contacter le service client d'Ubisoft Entertainment, merci de parcourir notre FAQ ainsi que la base de données d'assistance sur notre site : http://support.ubi.com. Vous y trouverez des informations mises à jour régulièrement depuis la sortie du jeu.

De même, assurez-vous que votre ordinateur possède au moins la configuration minimale, notre service client ne pourra pas aider les personnes qui ne remplissent pas ce critère.

Lorsque vous contactez le service client, veuillez renseigner les informations suivantes ou les avoir à portée si vous appelez :

-Nom complet du produit (dont le numéro de version)
-Contenu exact du message d'erreur (le cas échéant) et une brève description du problème que vous rencontrez
-Vitesse du processeur et nom de son fabricant
-Quantité de mémoire vive
-Système d'exploitation
-Carte graphique que vous utilisez et sa quantité de mémoire vidéo
-Modèle et vitesse de votre lecteur CD ou DVD
-Type de carte son utilisée
